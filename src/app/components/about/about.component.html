Welcome to my Spotify web application

Currently the app breaks due to developer access token expiring

This application was made using HTML, CSS, Javascript, Typescript, Angular and Bootstrap. This app uses the Spotify API to receive information about artists and albums. Searching for an artist will return any artist that matches the search term and clicking on an artist will show all albums that spotify possesses from the selected artist. Selecting an individual album will show all songs from that album. Selecting a song redirects the user to spotify for a sample clip from the selected song.
